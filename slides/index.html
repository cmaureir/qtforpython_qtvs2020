Setup 🔨

Installing PySide


    pip install pyside2

What does that install?


     ▪️PySide2                     ✅
      ├── Qt modules              ✅
      ├── pyside2-designer        ✅
      ├── pyside2-rcc             ✅
      └── pyside2-uic             ✅
     ▪️Shiboken2 (module)          ✅
      └── shiboken2               ✅
     ▪️Shiboken2_Generator (exe)   ❌
      └── shiboken2               ❌

Installing Shiboken Generator

the simple way


    pip install \
        --index-url=http://download.qt.io/official_releases/QtForPython/ \
        --trusted-host download.qt.io \
        shiboken2 pyside2 shiboken2_generator

But one needs to:

  • Set CLANG_INSTALL_DIR to the libclang directory
  • Add to PATH a Qt bin path with the same version
  • Add to LD_LIBRARY_PATH the Qt lib path with the same version

Installing Shiboken Generator

the hard way?

  • Set CLANG_INSTALL_DIR to the libclang directory

    python setup.py install
    # there are many other options!
